Elite Info Panel

An open-source WPF application for displaying real-time Elite Dangerous game information.

Download Latest Release

Features

  • Shows current ship data (hull, shields, modules)
  • Displays active missions and cargo manifests
  • Customizable panels and themes

Usage

  1. Download and extract the latest release.
  2. Run EliteInfoPanel.exe and connect to your Elite Dangerous API.
  3. Customize panels via the Settings menu.
